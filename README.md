 
"## API Endpoints" 
"- GET /lessons - Returns all lessons" 
 
"- POST /orders - Create new order" 
 
"- PUT /lessons/:id - Update lesson spaces" 
 
"- GET /search?q=term - Search lessons" 
 
"## Setup Instructions" 
"1. npm install" 
"2. Create .env file with MongoDB URI" 
"3. npm start to run server"
